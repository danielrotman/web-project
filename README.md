# 🌐 Web Development Project - Afeka College

פרויקט בניית אתר המציג את "5 האתרים היפים ביותר", שפותח במסגרת לימודי הנדסת תוכנה ב**אפקה - המכללה האקדמית להנדסה**.

## 🚀 Live Demo
**[לצפייה באתר החי לחצו כאן](https://danielrotman.github.io/web-project/)**

## 🛠 טכנולוגיות
* **HTML5** - מבנה סמנטי (Header, Nav, Sections).
* **CSS3** - עיצוב רספונסיבי וחווית משתמש.
* **GitHub Pages** - אירוח ופריסה של האתר.

## 📋 מה בפרויקט
* תצוגה של אתרים נבחרים עם ניווט מהיר.
* **מפת ניווט עולמית** (Image Map) אינטראקטיבית.
* התאמה מלאה למובייל ולדסקטופ (Responsive Design).
* שימוש בטפסים ותפריטים מעוצבים.
